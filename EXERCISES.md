## 💻 How to Use Each Exercise

1. **Open the exercise file** in your SQL editor (pgAdmin, DBeaver, DB Fiddle, etc.)
2. **Run the SETUP section** to create tables and populate seed data
3. **Attempt the questions** — try to write your own queries first
4. **Check solutions** only after you've tried
5. **Understand the WHY** — read the solution comments and explanations
6. **Try the challenge** — tackle optional challenge questions for deeper learning
