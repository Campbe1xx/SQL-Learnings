# Project 03 — Production Batch Report

## Objective
Analyse production order data to generate shift-level and machine-level performance reports, including running totals and output rankings.

## Skills Practiced
- Aggregates, GROUP BY, HAVING
- Window functions (RANK, SUM OVER, LAG)
- CTEs and PIVOT
- Date functions

## Extension Challenges
- Build a daily OEE report combining production output and downtime
- Compare each shift's output to the previous shift using LAG()
