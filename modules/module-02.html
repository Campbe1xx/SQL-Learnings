📚 Module 02 – Reading Data

Master data retrieval with SELECT, JOINs, aggregates, subqueries, and window functions

Beginner → Intermediate

📖 Lesson Overview

In this module, you'll learn:

  • SELECT, WHERE, ORDER BY, LIMIT – Basic data retrieval
  • Aggregates – SUM(), COUNT(), AVG(), MIN(), MAX()
  • GROUP BY & HAVING – Grouping and filtering aggregates
  • JOINs – INNER, LEFT, RIGHT, SELF joins
  • Subqueries & CTEs – Nested and common table expressions
  • Window Functions – ROW_NUMBER, RANK, LAG, LEAD, running totals
