# Project 04 — Quality & Defect Log

## Objective
Track quality inspection results per production batch, identify high-defect machines and operators, and flag batches that fail quality thresholds.

## Skills Practiced
- Schema design
- Aggregates and HAVING
- CASE statements
- Subqueries and CTEs

## Extension Challenges
- Write a view that shows a live quality dashboard
- Create a trigger that marks an order as 'Failed' when defect_rate > 5%
