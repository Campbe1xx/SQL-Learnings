🎯 Suggested Learning Paths

Path A: Complete Beginner

Start with basics and progress through all modules in order:

  1. exercises/basics/exercise_01.sql
  2. exercises/aggregations/exercise_01.sql
  3. Module 01 exercises
  4. Module 02 exercises (01–05)
  5. Continue through modules sequentially...

Path B: Fast Track (Prior SQL Experience)

Skip fundamentals and jump to advanced patterns:

  1. Module 02 exercises (04–05: Subqueries, Window Functions)
  2. Module 06 exercises (Views, Procedures, Triggers)
  3. Module 07 exercise (Performance)
  4. Module 09 capstone project

Path C: Topic-Focused

Pick your area of interest:

  • JOINs: exercises/joins/ + Module 02 exercise_03
  • Window Functions: exercises/window-functions/ + Module 02 exercise_05
  • Advanced Patterns: Module 06 exercises
  • Performance: Module 07 exercise
